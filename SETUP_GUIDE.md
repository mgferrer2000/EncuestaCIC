# Guía de Uso

Esta aplicación encuesta guarda los datos localmente en un archivo CSV.

## Dónde están los datos
Los datos se guardan en el archivo:
`server/data.csv`

Puedes abrir este archivo directamente con **Excel** o subirlo a Google Sheets cuando quieras analizar los datos.

## Cómo Iniciar
Simplemente ejecuta el archivo:
`encuesta.bat`

Esto abrirá dos ventanas:
1. El Servidor (Backend)
2. La Aplicación Web (Frontend)

¡Eso es todo! No se requiere configuración de Google Cloud.
